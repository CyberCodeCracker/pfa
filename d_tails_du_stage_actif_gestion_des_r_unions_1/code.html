Description du Projet

Ce stage se concentre sur l'exploration et le raffinement des architectures de réseaux neuronaux convolutifs (CNN) pour des applications de reconnaissance d'objets en temps réel sur des dispositifs embarqués. L'objectif est de réduire la latence tout en maintenant une précision élevée.

  • Analyse comparative des modèles MobileNetV3 et EfficientNet.
  • Mise en œuvre de techniques de quantification de poids (8-bit quantization).
  • Optimisation du pipeline de données via NVIDIA DALI.

Les étudiants travailleront en collaboration directe avec l'équipe de recherche en IA pour intégrer leurs découvertes dans le framework interne de l'institution.
